## Getting Started

- Install npm

```bash
npm install
```

- Run the server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
#or
Open [https://app-test-teravin.vercel.app](https://app-test-teravin.vercel.app) with your browser to see the result via online.
